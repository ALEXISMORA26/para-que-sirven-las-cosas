para levantar el revidor de angular: ng serve -o
para crear un nuevo proyecto: ng new "my-first-project"
para instalar todos los paquetes: npm i
para generar un servicio: ng g s (nombre del servicio)
para generar un componente: ng g c (nombre del componente)
para instalar el loader icon: npm i ngx-ui-loader
para instalar el jwt-decode: npm i jwt-decode
